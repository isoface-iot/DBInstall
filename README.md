## 簡介

資料庫安裝精靈(DBInstall)是愛招飛推出的輔助安裝工具，用於將愛招飛產品中所使用的示例資料庫掛載至資料庫伺服器中。在使用該產品前，請先在資料庫伺服器中安裝配置完成資料庫系統管理軟體的安裝。該安裝精靈只能在需要掛載資料庫的本機使用，不支援遠端掛載。
